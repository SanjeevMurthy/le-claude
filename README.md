# le-claude

A knowledge base and skill development repo for mastering Claude Code — covering setup, best practices, skills authoring, and AI-assisted workflows.

## What's in this repo

### `claude-tutorial/`
Comprehensive guides distilled from official docs, community posts, and video transcripts:
- **claude-code-guide.md** — End-to-end reference for Claude Code (install → agents → automation)
- **claude-code-webdesign-guide.md** — Building premium websites with Claude Code skills, MCP tools, and design workflows

### `data/`
Raw research material (video transcripts, blog excerpts) used to build the guides:
- `claude-setup-data/` — Claude Code setup, skills, agents, and best-practice transcripts
- `claude_website_design/` — Web design workflow transcripts

### `todo-skills/`
Design docs and specs for skills still in development:
- **interview-prep-doc.md** — Spec for an expert-level interview preparation skill
- **certificate-preparation-skill.md** — Spec for a certification exam study-guide skill

### `initialize.md`
Project brief outlining the goals and research plan for the documentation effort.

### `prompt.txt`
Additional prompt notes for skill requirements (interview prep enhancements).

## Getting started

1. Clone the repo:
   ```bash
   git clone git@github.com:SanjeevMurthy/le-claude.git
   ```
2. Open in your editor and point Claude Code at this directory.
3. Browse `claude-tutorial/` for the guides, or `todo-skills/` for upcoming skill specs.

## Contributing

- Add new research transcripts under `data/`.
- Draft new skill specs in `todo-skills/`.
- Keep guides in `claude-tutorial/` up to date as Claude Code evolves.
