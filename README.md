## Getting Started

1. **Clone the repo:**
   ```bash
   git clone git@github.com:SanjeevMurthy/le-ai-dev-guides.git
   ```

2. **Browse the guides:**
   - For Claude Code → start with [`guides/claude-code/claude-code-guide.md`](guides/claude-code/claude-code-guide.md)
   - For Antigravity IDE → start with [`guides/antigravity-ide/01-introduction.md`](guides/antigravity-ide/01-introduction.md)
   - For building full applications → start with [`guides/claude-code/building-with-claude-sop.md`](guides/claude-code/building-with-claude-sop.md)

3. **Explore upcoming skills** in `skills-in-progress/`
